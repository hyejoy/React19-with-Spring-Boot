# React 19 with Spring Boot 풀스택 스터디

React 19와 Spring Boot를 함께 학습하는 풀스택 개발 스터디 레파지토리입니다.

## 학습 목표

- React 19 + Redux Toolkit + React Query 실전 활용
- Spring Boot REST API 서버 구축
- JWT 인증, 소셜 로그인 구현
- 프론트엔드와 백엔드 연동 전체 흐름 이해

## 기술 스택

| 분류     | 기술                                             |
| -------- | ------------------------------------------------ |
| Frontend | React 19, Redux Toolkit, TanStack Query, Zustand |
| Backend  | Spring Boot, JPA, QueryDSL, MariaDB              |
| 인증     | JWT, Spring Security, 카카오 소셜 로그인         |
| 스타일   | Tailwind CSS                                     |
